## Becoming a Maintainer

Decision Kernel welcomes maintainers who:
- Demonstrate sustained, high-quality contributions
- Understand the kernel architecture deeply
- Commit to stability and neutrality
- Align with project philosophy

See [GOVERNANCE.md](GOVERNANCE.md) for details.

## Responsibilities

Maintainers are responsible for:
- Reviewing and merging pull requests
- Maintaining code quality standards
- Upholding architectural principles
- Ensuring test coverage
- Preserving API stability
- Maintaining vendor neutrality
